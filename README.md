## Getting Started
- Clone the Repository

```bash
git clone https://github.com/vishalparmarr/nextjs-auth-kindle.git
cd nextjs-auth-kindle
```
- Install Dependencies

```bash
npm install
```
- Change the env files
``` bash
.env.example to .env
and add the environment variables 
```

- Run the Server

```bash
npm run dev
```
